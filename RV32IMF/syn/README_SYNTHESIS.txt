The .sdc files are used by TimeQuest Analyzer to properly add pre and post-
module registers at the I/O terminals of each module name (eg. the core.sdc
is prepared to allow any core.v synthesis).

The sv_files_Quartus.qip is a TCL script to load all the components of the
project in one go.
